Pre-Workshop Scratch Tutorials

Complete these 2 tutorials before 5 October 2025

Why Complete These Tutorials?

These tutorials give you hands-on experience with Scratch before the workshop. You'll learn basic coding concepts that form the foundation for teaching Grade 4 learners.

Time Required: Each tutorial takes approximately 30-45 minutes. Set aside 2 hours total to complete both without rushing.

Tutorial 1: Space Talk

What you'll learn: Basic Scratch interface, sprites, backdrops, and simple animation

Step 1: Go to the Raspberry Pi Foundation website
Step 2: Create a free account (you'll need this to save your work)
Step 3: Follow the tutorial instructions step by step
Step 4: Save your completed project
Step 5: Take a screenshot of your finished project
Start Tutorial 1

Tutorial 2: Catch the Bus

What you'll learn: Movement, user input, and basic game mechanics

Step 1: Make sure you're logged into your Scratch account
Step 2: Follow the tutorial to create an interactive game
Step 3: Experiment with changing the code
Step 4: Save your project
Step 5: Take a screenshot showing your game working
Start Tutorial 2

Troubleshooting Tips

  • Can't see the tutorials? Make sure you're using a modern browser (Chrome, Firefox, Edge)
  • Scratch not loading? Check your internet connection or try a different device
  • Stuck on a step? Contact us through the Help page
  • Want to learn more? The Raspberry Pi site has many additional tutorials you can explore

After Completing the Tutorials

Take screenshots of both completed projects and keep them ready. You'll share these during the workshop to show your pre-work completion.
